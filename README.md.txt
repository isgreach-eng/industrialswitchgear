# Industrial Switchgear Website

This is the official website for **Industrial Switchgear**, hosted at [https://industrialswitchgear.in](https://industrialswitchgear.in).

The site is built using plain HTML, CSS, and JavaScript.

## Structure
- `index.html` → Main landing page
- `/css` → Stylesheets
- `/images` → Images & logos
- `/js` → Scripts (if any)

## Deployment
Hosted on **GitHub Pages** and mapped to the custom domain `industrialswitchgear.in`.
